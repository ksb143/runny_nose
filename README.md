# 코감기 알고리즘 스터디

## 방법

- 화, 목, 토 아침 스터디 (평일 8:40, 주말 9:00)

- 발표자 그 전날 Review 폴더에 푼 문제(주석 상세히) 업로드

- 나머지 스터디원들 숙지 하기

- 발표는 5분 이내로 짧게 발표 후 질문을 받기

- 지각, 문제 미제출자 벌금
