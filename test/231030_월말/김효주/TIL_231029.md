### REST API 월말평가 대비

#### REST API 란?

1. 식별 : URI (자원의 위치를 식별할 수 있는 식별자) 우리는 URL 을 사용하기로 하였음.

2. 행위 : HTTP Methods 로 구분 . 우리는 GET, POST, PUT, DELETE 4가지를 배움.

3. 표현 : 궁극적으로 표현되는 데이터 결과물. 다양한 언어, 환경에서 호환되어 사용될 수 있도록 함. 우리는 JSON 형식을 사용하기로 하였음.
